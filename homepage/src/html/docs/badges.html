Bootstrap documentation

Soft

Landkit only

Creates a soft variant of a corresponding contextual badge variation. These can be used exactly like Bootstrap's core badges, including modifying classes like .rounded-pill, as an <a> itself, or inside of <button> or <a> elements.

Primary Secondary Success Info Warning Danger Light Dark
<span class="badge bg-primary-soft">Primary</span>
<span class="badge bg-secondary-soft">Secondary</span>
<span class="badge bg-success-soft">Success</span>
<span class="badge bg-info-soft">Info</span>
<span class="badge bg-warning-soft">Warning</span>
<span class="badge bg-danger-soft">Danger</span>
<span class="badge bg-light-soft">Light</span>
<span class="badge bg-dark-soft">Dark</span>

Sizing

Landkit only

Create badges of a bigger size with a .badge-lg modifier.

Large Standard
<span class="badge badge-lg bg-primary">
  <span class="h6 text-uppercase fw-bold">Large</span>
</span>

Circle

Landkit only

Turn any badge into a circle with a .badge-rounded-cirle modifier.

<span class="badge badge-lg badge-rounded-circle bg-primary">
  <i class="fe fe-check"></i>
</span>

<span class="badge badge-rounded-circle bg-secondary-soft">
  <i class="fe fe-check"></i>
</span>

Float

Landkit only

Position any badge absolutely inside or outside an element with .badge-float-inside and .badge-float-outside modifiers.
